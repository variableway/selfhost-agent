# 安装 Node.js

> 适用人群：前端开发初学者
> 预计时间：10 分钟

## 什么是 Node.js？

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行时，让你能在服务器端运行 JavaScript。我们将使用 fnm（Fast Node Manager）来安装和管理 Node.js 版本。

## 前置条件

- 安装了终端工具
- 有稳定的网络连接

## 安装 fnm

fnm 是一个快速、简单的 Node.js 版本管理器：

<RunButton command="curl -fsSL https://fnm.vercel.app/install | bash" />

安装完成后，重新打开终端或运行以下命令使 fnm 生效：

```bash
source ~/.bashrc  # 或 source ~/.zshrc
```

## 安装 Node.js

使用 fnm 安装最新 LTS 版本：

<RunButton command="fnm install --lts" />

## 验证安装

确认 Node.js 和 npm 已正确安装：

<RunButton command="node -v && npm -v" />

如果看到版本号输出，说明安装成功。

## 版本管理

切换 Node.js 版本：

```bash
fnm list          # 查看已安装的版本
fnm use 20        # 切换到指定版本
fnm default 20    # 设置默认版本
fnm install 22    # 安装其他版本
```

## 完成检查

- [ ] fnm 已安装
- [ ] Node.js LTS 已安装
- [ ] `node -v` 和 `npm -v` 输出正确版本号
