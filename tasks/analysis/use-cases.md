# Executable Tutorials Uses Cases

## UC1: Upload the markdown or mdx file 

1. User create a topic,series lesson with name,cateogry.
2. User create the topic name, category name ,and upload the md,mdx file in this series lesson/Topic
3. In the lesson page, use can see this section which is the file
4. user can add another similar file into this series lesson,then the file will be displayed in the lesson page
5. User can change the order of the files in the series lesson, the order refect the display order in the tutorial page
6. User can preview the tutorial page in the application, preview one page after another, if there is multiple sections in this lesson, there is a next button to navigate to the next section based on the order set
