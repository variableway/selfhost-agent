# Local AI Playground

目前想要实现一个本地AI的Playground。
用户主要是：
1. 几乎没有编程程序经验的小白
2. 只有一点点基础的小白但是有一定基础的用户，比如大学本科毕业
用户可以用来做什么：
1. 按照教程一步一步搭建自己本地使用AI的API调用
2. 按照教程使用AI的API调用
3. 按照教程使用AI编写一些程序运行
4. 可以使用别人用的日常效率工具
5. 可以构建自己日常效率工具

这个工具能够做到什么：
1. 基本内容一键安装配置
2. UI帮助用户配置好环境
3. 提供一些基础支持的教程说明，已经一些ROADMAP内容
4. 提供一些基础的日常效率工具，比如AI助手、AI翻译、AI图片生成等，复杂一点也许使用cli去操作browser完成这些日常工具的使用

短期先用OpenClaw类似这样的东西为例子，帮可以一键安装说明，哪怕是各种云claw的一个门户。
还有就是可以本地安装使用的代码相关工具的安装，使用，以及一些基本教程，以及IDE的使用教程
- 一些可视化的基本说明，比如UI的Layout，组件的使用说明，然后如何使用这些组件能让AI做成什么样东西
- 一些ChatDashBoard到的组件，然后让客户感觉可以做什么

整个形式：
1. web端，但是无法操作本地机器
2. 本地端可以直接操作本地机器，一步一步处理，同时把terminal内容也展示出来

目标是轻量级给个人使用。所有的内容都需要写入到新的文档，可以多个文档。比如分析，架构设计，计划这些不同的维度的文档。
