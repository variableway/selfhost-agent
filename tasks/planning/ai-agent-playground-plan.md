# AI Agent Playground Desktop Application — 项目计划

> 项目代号：Innate Playground
> 创建时间：2026-04-06
> 状态：待 Review

## 一、项目目标

为**非技术用户**构建一个桌面应用，使其能在 5-10 分钟内完成 AI Agent 相关工具的安装和学习。

### 核心价值

1. **零门槛安装** — 一键安装所有依赖，无需手动操作终端
2. **可视化学习路径** — RoadMap 引导用户从入门到进阶
3. **内置终端** — 在应用内直接看到安装过程，无需切换工具
4. **短教程体系** — 每个教程 5-10 分钟，配有脚本和视频

### 目标用户

- 对 AI Agent 感兴趣但不会编程的用户
- 安装环境经常失败的新手
- 想快速体验 AI 工具但不想折腾配置的人

## 六、风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| PTY 集成复杂 | 高 | 先用 Tauri shell 插件的简单模式，后续升级 PTY |
| Windows 权限问题 | 中 | 应用 manifest 请求管理员权限；优先使用无需 sudo 的方案 |
| 中文路径/编码 | 低 | 路径使用 UTF-8，测试中文用户场景 |
| Tauri v2 插件不成熟 | 低 | 核心 shell 插件已稳定，社区活跃 |

---

## 七、成功指标

- 新用户 30 分钟内完成首个教程（安装终端工具）
- 5 分钟内完成一个 AI CLI 工具的安装配置
- 安装包 < 30MB（Mac + Windows）
- 应用内终端能完整显示安装过程
