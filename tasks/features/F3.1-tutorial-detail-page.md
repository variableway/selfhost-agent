# F3.1 教程详情页

## Description

实现教程详情页面，包含教程信息、步骤列表、内置终端和操作按钮。

## Acceptance Criteria

- [ ] 展示教程标题、描述、预计时间、难度
- [ ] 展示步骤列表，每步显示：序号、标题、描述、状态图标
- [ ] 步骤状态实时更新：待执行 → 执行中 → 成功/失败
- [ ] 内嵌终端面板，实时显示当前命令输出
- [ ] 底部操作栏：一键安装 / 执行下一步 / 重试 / 跳过
- [ ] 教程完成后显示成功动画和下一步推荐

## Technical Notes

- 步骤列表使用 shadcn/ui 的 Accordion 或自定义组件
- 终端面板可折叠/展开
- 使用 Zustand store 管理教程执行状态

## Dependencies

- F1.3 应用布局与导航
- F2.2 xterm.js 终端组件
- F2.3 实时输出通信
- F2.4 步骤执行引擎
