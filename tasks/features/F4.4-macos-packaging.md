# F4.4 macOS 打包 (.dmg)

## Description

配置 Tauri 的 macOS 打包流程，生成可分发的 .dmg 安装包。

## Acceptance Criteria

- [ ] `pnpm tauri build` 生成 .dmg 文件
- [ ] 安装包大小 < 30MB
- [ ] 应用图标和名称正确
- [ ] 安装后能在 Applications 中找到
- [ ] 应用签名（如有 Apple Developer 账号）

## Technical Notes

```json
// tauri.conf.json
{
  "bundle": {
    "active": true,
    "targets": ["dmg"],
    "icon": ["icons/icon.icns"],
    "identifier": "com.innate.playground",
    "category": "Education",
    "macOS": {
      "minimumSystemVersion": "10.15"
    }
  }
}
```

## Dependencies

- Phase 1 + Phase 2 + Phase 3 完成

## Estimated Time

2-3 小时

## Difficulty

中
