# F1.5 教程列表页面

## Description

实现教程列表视图，按分类展示所有教程，支持筛选和搜索。

## Acceptance Criteria

- [ ] 所有教程以列表/网格形式展示
- [ ] 支持按难度筛选（全部 / 入门 / 进阶 / 高级）
- [ ] 支持按标签筛选（terminal, nodejs, python, ai-cli 等）
- [ ] 支持搜索教程名称和描述
- [ ] 每个教程卡片显示：名称、描述、难度徽章、预计时间、平台图标
- [ ] 已完成的教程有视觉区分
- [ ] 点击进入教程详情页
