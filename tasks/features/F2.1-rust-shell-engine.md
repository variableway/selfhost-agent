# F2.1 Rust Shell 命令执行层

## Description

在 Tauri Rust 后端实现 Shell 命令执行引擎，支持命令执行、输出流式传输、进程管理。

## Acceptance Criteria

- [ ] 实现 `execute_command` — 执行命令并返回完整输出
- [ ] 实现 `spawn_command` — 启动命令并通过 Tauri 事件流式输出
- [ ] 实现 `kill_command` — 终止指定 session 的命令
- [ ] stdout/stderr 通过 Tauri 事件实时推送到前端
- [ ] 支持设置工作目录 (cwd)
- [ ] 支持环境变量传递
- [ ] 正确处理跨平台差异（macOS 用 bash/zsh，Windows 用 PowerShell）

### 跨平台注意

```rust
fn get_shell_command() -> (&'static str, Vec<&'static str>) {
    if cfg!(target_os = "windows") {
        ("powershell", vec!["-NoProfile", "-Command"])
    } else {
        ("bash", vec!["-c"])
    }
}
```
