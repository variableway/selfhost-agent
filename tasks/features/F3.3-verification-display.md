# F3.3 验证步骤展示

## Description

实现安装验证功能，在安装完成后自动运行验证命令，展示验证结果。

## Acceptance Criteria

- [ ] 支持 Skill JSON 中 `type: "verification"` 的步骤
- [ ] 执行验证命令并比对 `expected_output`
- [ ] 验证成功显示绿色勾选 + 工具版本号
- [ ] 验证失败显示红色叉 + 具体差异
- [ ] 在 RoadMap 首页显示已安装工具的版本信息

## UI Design

```
验证安装结果:

  ✅ Homebrew   v4.2.0
  ✅ iTerm2     已安装
  ✅ Zsh        v5.9
  ✅ Oh My Zsh  已安装
  ✅ tmux       v3.4
  ✅ fzf        v0.45.0
  ✅ bat        v0.24.0
  ❌ ripgrep    未找到

  [重新验证]  [重新安装 ripgrep]
```

## Technical Notes

验证命令示例：
- `which brew && brew --version` → 检查 Homebrew
- `node -v` → 检查 Node.js
- `python3 --version` → 检查 Python

比对逻辑：验证命令的 exit code == 0 即视为成功，无需严格比对输出文本。
