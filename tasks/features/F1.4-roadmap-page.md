# F1.4 首页 RoadMap 页面

## Description

实现首页 RoadMap 视图，以可视化学习路径展示所有教程，包括进度、依赖关系和推荐顺序。

## Acceptance Criteria

- [ ] 读取 `skills/` 目录下所有 Skill JSON 文件
- [ ] 按难度分组展示（beginner / intermediate / advanced）
- [ ] 每个教程显示为卡片：名称、描述、预计时间、平台支持、进度状态
- [ ] 显示教程间的依赖关系（prerequisites）
- [ ] 已完成的教程显示勾选标记
- [ ] 不可用的教程（前置条件未满足）显示为灰色
- [ ] 点击教程卡片跳转到教程详情页

## Technical Notes

- Skill JSON 从 Tauri 后端读取（打包进应用资源）
- 进度数据从 localStorage 或文件持久化读取
- 根据当前平台过滤教程（macOS 只显示支持 macOS 的教程）
