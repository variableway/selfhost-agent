# F3.2 一键安装流程

## Description

实现一键安装功能，自动按顺序执行 Skill JSON 中所有步骤，带进度条和错误处理。

## Acceptance Criteria

- [ ] 点击"一键安装"自动执行所有步骤
- [ ] 顶部显示总体进度条（3/5 步骤已完成）
- [ ] 每步自动切换，终端面板实时显示当前命令输出
- [ ] 步骤间有短暂过渡动画
- [ ] 某步失败时自动暂停，显示错误信息和重试选项
- [ ] 全部完成后显示成功页面
- [ ] 支持中途取消（kill 所有运行中的进程）

## Technical Notes

```typescript
async function executeAll(skill: Skill) {
  setTutorialStore({ isRunning: true });

  for (let i = 0; i < skill.steps.length; i++) {
    setTutorialStore({ currentStepIndex: i });

    try {
      const result = await invoke('execute_step', {
        step: skill.steps[i],
        platform: platformStore.os,
      });

      addStepResult(i, result);

      if (result.status === 'failed') {
        // 暂停，等待用户操作
        setTutorialStore({ isPaused: true, failedAtIndex: i });
        return;
      }
    } catch (error) {
      // 同上
    }
  }

  // 全部完成
  setTutorialStore({ isRunning: false, completedAt: Date.now() });
  markSkillCompleted(skill.id);
}
```
