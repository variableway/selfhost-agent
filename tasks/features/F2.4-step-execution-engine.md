# F2.4 步骤执行引擎

## Description

实现教程步骤执行引擎，按 Skill JSON 中定义的 steps 依次执行命令，管理步骤状态和依赖。

## Acceptance Criteria

- [ ] 能解析 Skill JSON 的 steps 数组
- [ ] 依次执行每个步骤（按 order 排序）
- [ ] 支持 3 种步骤类型：
  - `script` — 自动执行命令
  - `verification` — 执行验证命令，比对 expected_output
  - `manual` — 显示提示，等待用户手动确认
- [ ] 步骤失败时暂停，支持重试
- [ ] 步骤执行过程中实时更新 UI 状态
- [ ] 支持跳过某些步骤（用户选择）

## Technical Notes

```typescript
// lib/step-executor.ts
interface StepExecutor {
  execute(skill: Skill, platform: string): AsyncGenerator<StepEvent>;
}

type StepEvent =
  | { type: 'step:start'; index: number; step: Step }
  | { type: 'step:output'; index: number; data: string }
  | { type: 'step:complete'; index: number; result: StepResult }
  | { type: 'step:failed'; index: number; error: string }
  | { type: 'skill:complete'; results: StepResult[] }
  | { type: 'skill:paused'; failedAtIndex: number };
```
