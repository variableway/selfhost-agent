# F4.3 视频教程集成

## Description

在教程详情页中集成视频播放功能，支持外部视频链接。

## Acceptance Criteria

- [ ] 教程详情页支持视频区域
- [ ] 支持嵌入外部视频（Bilibili / YouTube iframe）
- [ ] 视频和终端面板可并排显示或切换
- [ ] Skill JSON 中添加 `video_url` 字段

## Technical Notes

Skill JSON 扩展：

```json
{
  "video_url": "https://www.bilibili.com/video/BVxxx",
  "resources": [
    { "type": "video", "url": "https://...", "title": "视频教程" },
    { "type": "doc", "url": "docs/guides/...", "title": "文档" }
  ]
}
```

## Dependencies

- F3.1 教程详情页

## Estimated Time

2-3 小时

## Difficulty

低
