# F4.5 Windows 打包 (.exe/.msi)

## Description

配置 Tauri 的 Windows 打包流程，生成可分发的安装包。

## Acceptance Criteria

- [ ] `pnpm tauri build` 生成 .exe 或 .msi 安装包
- [ ] 安装包大小 < 30MB
- [ ] 应用图标和名称正确
- [ ] 安装后能在开始菜单和桌面找到
- [ ] PowerShell 脚本正确打包到应用资源中

## Technical Notes

```json
// tauri.conf.json
{
  "bundle": {
    "targets": ["msi", "nsis"],
    "windows": {
      "webviewInstallMode": { "type": "downloadBootstrapper" }
    }
  }
}
```

### 注意事项

- Windows 需要 WebView2 Runtime（Tauri 可自动下载安装）
- PowerShell 脚本的执行策略可能需要 bypass
- 安装路径避免中文和空格

## Dependencies

- Phase 1 + Phase 2 + Phase 3 完成

## Estimated Time

3-4 小时
