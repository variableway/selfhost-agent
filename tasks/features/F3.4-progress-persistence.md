# F3.4 进度持久化

## Description

实现用户学习进度和安装状态的持久化存储。

## Acceptance Criteria

- [ ] 已完成的教程 ID 持久化保存
- [ ] 每个教程的步骤完成状态持久化
- [ ] 已安装工具及版本号缓存
- [ ] 应用重启后恢复之前的进度
- [ ] 支持"重置进度"功能

## Technical Notes

### 存储方案

使用 Tauri 的 `app_data_dir` 存储进度 JSON 文件：

```rust
// 存储路径: ~/.local/share/innate-playground/progress.json (macOS)
//           %APPDATA%/innate-playground/progress.json (Windows)

#[tauri::command]
async fn save_progress(progress: UserProgress) -> Result<(), String> {
    let path = app_data_dir().join("progress.json");
    fs::write(path, serde_json::to_string(&progress)?)?;
    Ok(())
}

#[tauri::command]
async fn load_progress() -> Result<UserProgress, String> {
    let path = app_data_dir().join("progress.json");
    if path.exists() {
        let data = fs::read_to_string(path)?;
        Ok(serde_json::from_str(&data)?)
    } else {
        Ok(UserProgress::default())
    }
}
```

### 数据结构

```json
{
  "completedSkills": ["terminal-setup-mac", "nodejs-setup"],
  "skillProgress": {

}
```
