# 小白 10 分钟速赢指南：复制即用 Prompt 合集

> 无需代码、无需配置、无需 API，只要会打开 ChatGPT / Claude / Kimi / 豆包，复制粘贴就能出效果。
> 分析日期：2026-04-16

---

## 筛选逻辑：为什么是这 6 个？

| 标准 | 说明 |
|------|------|
| **零门槛** | 不需要安装软件、不需要写代码、不需要配置文件 |
| **零成本** | 用任意免费版大模型（网页/APP）就能跑 |
| **立即可见** | 输入 → 10 秒内出结果，结果可以直接保存/使用 |
| **高频刚需** | 每周甚至每天都会遇到的真实场景 |

**速赢组合**：摄影诊断 + 三餐规划 + 打包清单 + 会议纪要 + 简历优化 + 周报生成

## 1. 摄影技术诊断（有图就能跑）

**适用**：手机拍了照片想知道拍得怎么样、怎么修更好。

**使用方式**：
1. 打开任意支持图片输入的 AI（ChatGPT、Claude、Kimi、Gemini）
2. 粘贴下面的 Prompt
3. 上传你的照片
4. 发送

### Prompt

```
你是一位专业的摄影教练。请对这张照片进行全面技术诊断，并按以下结构输出：

## 📷 基础信息
- 如果你能看到 EXIF，请列出：相机/手机型号、光圈、快门、ISO、焦距
- 如果没有 EXIF，请说明"无元数据"

## 🔍 四维度分析
### 1. 构图
评价画面的构图方式（三分法、对称、引导线、框架等），指出优点和改进点。

### 2. 曝光与光影
分析亮度分布、高光/阴影细节、对比度、色温是否合适。

### 3. 对焦与清晰度
评价主体是否清晰、景深控制是否得当、是否有手抖/失焦。

### 4. 色彩与氛围
分析配色、饱和度、情绪表达是否统一。

## 🛠️ 后期修图建议
给出 3-5 条具体的 Lightroom/手机修图 App 调整建议（用"参数名 + 数值"格式）。

## 🌟 一句话总结
用一句话概括这张照片最突出的优点和最大的改进空间。

请用中文回答，避免过多专业黑话，让摄影小白也能看懂。
```

### 效果示例
输出会是一个结构化的摄影点评报告，小白可以直接按照"后期修图建议"里的参数在手机 Lightroom/美图秀秀/VSCO 里尝试调整。

### 进阶玩法
如果你想批量筛图，可以用简化版：
```
请快速点评这张照片：优点是？最大问题是？给一条最关键的修图建议。
```

## 2. 明日三餐规划（改几个食材名就能用）

**适用**：每天晚上发愁明天吃什么，想吃得健康又不想动脑。

**使用方式**：
1. 复制 Prompt
2. 把【】里的内容换成你冰箱里的实际食材
3. 发送给任意大模型

### Prompt

```
请根据以下信息，帮我规划明天的三餐，并生成购物清单：

## 冰箱现有食材
【鸡蛋 5 个、鸡胸肉 300g、西兰花 1 颗、番茄 4 个、土豆 2 个、牛奶 1L、燕麦片、大米、冷冻虾仁 200g、胡萝卜 3 根】

## 我的饮食偏好/限制
【少油少盐、不吃香菜、早餐要快（10 分钟内）、午餐是便当需要微波炉加热、晚餐在家做】

## 营养目标
【减脂期，每天总热量控制在 1500-1700 kcal，蛋白质尽量高】

请输出以下内容：
1. 三餐详细方案（菜名 + 大致做法 + 预计耗时）
2. 每餐的热量估算和三大营养素占比
3. 基于现有食材，还需要买什么？列出购物清单
4. 标注哪些食材是冰箱已有的（✅），哪些需要购买（🛒）

用 Markdown 表格呈现，清晰易读。
```

### 效果示例
输出是一个可以直接截图发给买菜 App 或发给家人的完整菜单和清单。

### 进阶玩法
如果你连冰箱有什么都懒得写，可以拍一张冰箱内部照片，用多模态模型：
```
这是我冰箱现在的样子，请帮我看看有哪些食材，然后规划明天的三餐。
```

## 3. 差旅打包清单（改目的地和天数就行）

**适用**：临时要出差/旅行，总是忘带东西，想一键生成清单。

**使用方式**：
1. 复制 Prompt
2. 把【】里的信息改成你的行程
3. 发送给 AI

### Prompt

```
请帮我生成一份差旅打包清单。

## 行程信息
- 目的地：【上海】
- 天数：【3 天】
- 日期：【2026 年 4 月 20 日 - 4 月 22 日】
- 行程性质：【商务出差，有 1 场正式会议】
- 性别/着装风格：【男生，商务休闲】

## 输出要求
1. 查询目的地期间的天气预报（温度、降雨概率），并据此推荐衣物
2. 生成分类打包清单，带 □ 勾选框
3. 分类包括：衣物、洗漱用品、电子设备、证件文件、药品、其他
4. 根据行程性质推荐必须带的特殊物品（如名片、正装等）
5. 给出 2-3 条特别提醒（如"周二有雨记得带伞"）

请用 Markdown 输出，清单格式要可以直接复制到备忘录或打印。
```

### 效果示例
输出是一个带 □ 的清单，你可以直接复制到 iPhone 备忘录 / 飞书文档 / Notion 里，出门前逐项勾选。

### 进阶玩法
如果你有固定差旅偏好，可以把这个 Prompt 保存下来，每次只改目的地和日期，30 秒出一份新清单。

## 4. 会议纪要整理（录音转文字后一键结构化）

**适用**：开会时记录了杂乱的文字或语音转写稿，想快速整理成专业纪要。

**使用方式**：
1. 把会议录音用飞书妙记/讯飞听见/通义听悟转成文字
2. 复制文字稿
3. 粘贴下面的 Prompt + 文字稿
4. 发送

### Prompt

```
请将以下会议录音转写稿整理成一份专业的会议纪要。

## 整理要求
1. **会议基本信息**：提取会议主题、时间、参与人（如果文稿中有提到）
2. **会议议程**：列出本次会议讨论的几个主要议题
3. **关键结论**：用 bullet points 列出达成了哪些共识/决定
4. **待办事项（Action Items）**：用表格呈现，包含：事项、负责人、截止时间、优先级
5. **待确认/待跟进问题**：列出会上提出但还没解决、需要后续跟进的问题
6. **一句话总结**：用 1-2 句话概括本次会议的核心成果

## 格式要求
- 使用 Markdown 格式
- 待办事项必须用表格，且要有明确的负责人和截止时间（如果原文没有，标注"待确认"）
- 删除口语化表达、重复内容和与会议无关的闲聊
- 保留关键数据和数字

---

【在这里粘贴你的会议文字稿】
```

### 效果示例
输出是一个可以直接发到工作群或存档的正式会议纪要，领导看了会很满意。

### 进阶玩法
如果你经常主持固定例会，可以保存这个变体：
```
请将以下周会文字稿整理成周报格式的纪要，重点突出本周进展、下周计划和阻塞项。
```

## 5. 简历优化 / 面试准备（复制简历就能用）

**适用**：想跳槽但简历写得不够好，或者面试前想快速准备常见问题的回答。

### 5.1 简历优化 Prompt

```
你是一位资深 HR 和猎头顾问。请帮我优化以下简历内容。

## 原始简历内容
【在这里粘贴你的简历某一段内容，比如：
"负责公司公众号运营，写文章，策划活动，粉丝涨了。"
】

## 优化要求
1. 用 STAR 法则（情境-任务-行动-结果）重构描述
2. 尽量加入可量化的数据和指标
3. 使用更专业、更有行动力的动词开头
4. 突出与【目标岗位：新媒体运营】相关的技能和经验
5. 输出优化后的版本，并说明每一处改进的原因
```

### 5.2 面试押题 Prompt

```
我即将面试【产品经理】岗位，目标公司是【字节跳动】。

请基于以下我的简历内容，帮我预测 5 个最可能被问到的问题，并为每个问题给出建议回答框架（不是逐字稿，是回答思路和要点）。

## 我的简历亮点
【粘贴简历中的 2-3 个核心项目/经历】

## 输出格式
对每个问题给出：
1. 面试官为什么问这个问题（考察点）
2. 建议回答结构（分 1-2-3 点）
3. 避坑提醒（不要说什么）
```

### 效果示例
- 简历从口语化变成专业、数据化的表达
- 面试问题命中率通常能达到 60-80%

## 6. 周报 / 日报生成（罗列工作内容一键格式化）

**适用**：每周五下午不想花时间写周报，只想罗列几条工作内容让 AI 帮忙润色。

### 日报 Prompt

```
请帮我把以下的工作内容整理成一份专业的日报。

## 原始内容
【
- 早上参加了需求评审会
- 下午写了用户增长方案的初稿
- 跟设计确认了首页改版的效果图
- 回复了几个客户的邮件
】

## 输出要求
1. 用"今日完成"、"明日计划"、"遇到的问题与风险"三个模块呈现
2. 每句话都用更有行动力和结果导向的方式重写
3. 总字数控制在 200 字以内
4. 用 Markdown 输出
```

### 周报 Prompt

```
请帮我把本周的工作内容整理成一份结构清晰的周报。

## 本周工作
【
1. 完成了用户增长方案 V1.0 的撰写并提交给上级
2. 参加了 3 场跨部门会议（产品、技术、运营）
3. 处理了 15 个客户反馈，其中 3 个升级为 Bug 已转给技术
4. 调研了 2 个竞品的新功能并做了对比分析
5. 周五下午做了团队内部分享
】

## 输出要求
1. **本周进展**：按项目分类，突出成果和数据
2. **下周计划**：列出 3-5 条具体的待办事项
3. **问题与风险**：列出需要上级支持或跨部门协调的事项
4. **个人思考/建议**（可选）：从本周工作中提炼出的 1-2 条洞察
5. 语气专业但不生硬，适合直接发给直属 Leader
```

### 效果示例
原本像流水账的罗列，变成了可以直接提交的结构化工作汇报，节省 80% 的排版和措辞时间。

## 给小白的使用建议

1. **选一个你最痛的场景先试**：不要 6 个一起用，先挑一个今天就能用到的
2. **把 Prompt 存到备忘录/飞书文档里**：下次用的时候复制 + 改几个关键词就行
3. **多轮对话优化**：第一版输出如果不满意，直接告诉 AI "再详细一点" 或 "换个风格"
4. **不要迷信 AI**：摄影建议、营养估算、天气信息都需要你结合实际判断
